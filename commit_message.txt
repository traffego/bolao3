debug: Implementada bolha de debug sempre ativa para monitorar códigos de afiliação

- Criado debug_bolha.php: bolha visual que exibe códigos em tempo real
- Monitora: GET[ref], POST[referral_code], SESSION[referral_code], dados do banco
- Integrado ao header.php para aparecer automaticamente em todas as páginas
- Bolha sempre visível para monitoramento contínuo da persistência
- Interface visual com cores para identificar problemas rapidamente
- Objetivo: verificar se códigos de referência se mantêm durante toda a sessão
- Facilita identificação imediata de onde os códigos se perdem
