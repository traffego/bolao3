debug: Implementado debug simples e direto para monitorar códigos de afiliação

- Substituída bolha complexa por debug simples com echo no header.php
- Exibe claramente: GET[ref], SESSION[referral_code], dados do banco
- Debug sempre visível em todas as páginas de forma bem destacada
- Interface simples com fundo preto e texto verde para máxima visibilidade
- Mostra código do afiliado e referência de indicação do usuário logado
- Objetivo: verificar se códigos de referência se mantêm durante toda a sessão
- Facilita identificação imediata de onde os códigos se perdem
