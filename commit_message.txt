debug: Implementado sistema de debug no cadastro real para monitorar processo

- Criado debug_cadastro_real.php com logs detalhados de cada etapa
- Integrado ao cadastro.php para capturar o fluxo real do usuário
- Debug intercepta: GET/POST, sessão, validação, inserção no banco
- Função debugDbInsert substitui dbInsert para monitorar dados salvos
- Logs salvos em debug_cadastro_log.txt para análise posterior
- Parâmetros: ?debug=1 para exibir na tela, ?show_log=1 para ver log
- Objetivo: identificar exatamente onde ref_indicacao se perde no processo real
